# Cerebras CLI Utility (cerebrasllm)

Ультрабыстрая консольная утилита для взаимодействия с API **Cerebras Cloud AI** (LPU Inference Engine) через командную строку Windows. Обеспечивает колоссальную скорость генерации благодаря специализированным чипам Cerebras и поддерживает мультимодальный режим (включая работу с изображениями через `gemma-4-31b`).

## ✨ Возможности

*   **Реактивная скорость**: Построена на чипах LPU от Cerebras, обеспечивающих скорость генерации в тысячи токенов в секунду.
*   **Мультимодальность**: Поддержка текста и изображений (зрения) через модель Google `gemma-4-31b`.
*   **Совместимость с экосистемой**: Утилита использует ту же структуру истории чатов (`mistral_chats`), что позволяет бесшовно переключаться между Mistral, Gemini, Groq и Cerebras без потери контекста.
*   **Автоматическая ротация ключей**: Динамически перебирает пулл ключей из конфигурации для обхода лимитов.
*   **Поддержка Windows**: Корректная обработка кодировок (CP866, CP1251 -> UTF-8) консоли при работе через пайпы (pipes).

## 🛠 Сборка

Для сборки требуется установленный [Go 1.25+](https://go.dev/dl/).

```bash
# Перейдите в папку проекта
cd cmd/cerebrasllm

# Соберите исполняемый файл
go build -o cerebrasllm.exe
```

## ⚙️ Настройка

Утилита хранит настройки в файле `%AppData%\clipgen-m\cerebras.conf`.

**Добавить API ключ:**
Получите бесплатный или платный ключ в [Cerebras Cloud Console](https://cloud.cerebras.ai/).
```powershell
cerebrasllm.exe --save-key ccl-ваш_ключ
```
Вы можете добавлять несколько ключей. Утилита будет ротировать их случайным образом для распределения нагрузки.

## 🚀 Примеры использования

### 1. Простой текстовый запрос (по умолчанию Gemma 4)
```powershell
echo "Объясни квантовую физику простыми словами" | cerebrasllm.exe
```

### 2. Сложные задачи кодинга (модель gpt-oss-120b)
```powershell
echo "Напиши конкурентный воркер-пул на Go" | cerebrasllm.exe -m code
```

### 3. Анализ изображений и зрение (модель gemma-4-31b)
```powershell
echo "Что изображено на этом графике?" | cerebrasllm.exe -f "C:\Data\chart.png"
```

### 4. Режим чата (с сохранением истории)
```cmd
echo "Привет, меня зовут Дмитрий" | cerebrasllm.exe -chat session_01
echo "Как меня зовут?" | cerebrasllm.exe -chat session_01
```

## 📚 Справочник аргументов

| Флаг | Описание | Пример |
| :--- | :--- | :--- |
| `-f` | Путь к файлу (картинка или текст). Можно указывать несколько раз. | `-f "1.png"` |
| `-s` | Системный промпт (инструкция роли). | `-s "Отвечай как пират"` |
| `-j` | JSON режим. Форсирует ответ в валидном JSON. | `-j` |
| `-m` | Принудительный режим: `auto`, `general`, `code`, `vision`. | `-m vision` |
| `-t` | Температура (0.0 - 2.0). | `-t 0.7` |
| `-chat`| Идентификатор чата для сохранения контекста. | `-chat session_1` |
| `-v` | Verbose. Вывод подробных логов в stderr и файл. | `-v` |
| `--save-key`| Сохранить API ключ в конфиг и выйти. | `--save-key ccl-...` |

## 🧠 Логика работы с моделями

### Иерархия моделей
- **General**: `gemma-4-31b` -> `gpt-oss-120b` -> `zai-glm-4.7`
- **Code**: `gemma-4-31b` -> `gpt-oss-120b` -> `zai-glm-4.7`
- **Vision**: `gemma-4-31b` (на данный момент только Gemma 4 поддерживает мультимодальность)

## 📁 Файлы и логи
*   **Конфиг**: `%AppData%\clipgen-m\cerebras.conf`
*   **Логи**: `%AppData%\clipgen-m\cerebras_err.log`
*   **История чатов**: `%AppData%\clipgen-m\mistral_chats\` (общая с другими модулями).
